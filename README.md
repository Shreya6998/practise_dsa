# DSA Practice Repository

This repository contains my implementations of **Data Structures and Algorithms** problems in Java.

The goal of this repository is:

* Practice and revise core DSA concepts
* Organize problems by **pattern and data structure**
* Track **revision progress** of previously solved problems

---

# Repository Structure

```
Algorithms/          → Pattern-based algorithm problems
Data Structures/     → Problems grouped by underlying data structure
dsaProblems/         → Older problems due for revision
grokking_algorithms/ → Implementations from "Grokking Algorithms"
```

Problems in **`dsaProblems`** are older implementations that I plan to **re-solve and move to the appropriate category** after revision.
